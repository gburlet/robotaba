About


Robotaba is a guitar tablature transcription framework, in which polyphonic transcription and guitar tablature arrangement algorithms can be embedded. The framework is open source and distributed under the MIT license. The source code can be found here. Please feel free to contribute to the project or leave a bug report.

For more information refer to the Master's thesis:
Burlet, G. 2013. Automatic guitar tablature transcription online. Master's thesis, McGill University.
